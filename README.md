### 🔧 Recent Work

- **Cross-role authentication bug** — traced a production auth/routing failure to two root causes (a route guard blocking valid navigation, and shared `localStorage` causing session collisions across tabs); rebuilt session handling to fix both.
- **Two-stage email notification pipeline** — designed a Thymeleaf-based system separating OPS-assignment emails from journey-event emails, with Google Calendar sync via a backend service account.
- **Role-differentiated API rate limiting** — implemented Bucket4j-based limiting with separate thresholds for admin, general, and authentication traffic, closing a gap in undifferentiated request controls.
- **Chat/notification system rebuild** — replaced a fragile global-variable notification router with a reactive `BehaviorSubject`-based service, fixing push-notification deep linking and eliminating duplicate listener memory leaks.
